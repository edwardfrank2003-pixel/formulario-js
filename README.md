# Formulario Web con JavaScript

## Descripción
Este proyecto consiste en el desarrollo de un formulario web utilizando HTML, CSS y JavaScript. Incluye validación de datos, uso de eventos del DOM y simulación de envío mediante promesas.

## Funcionalidades
- Validación de campos obligatorios
- Validación de correo electrónico
- Mensajes de error visibles
- Uso de eventos DOM (submit)
- Simulación de envío con Promise (éxito y error)


Tecnologías utilizadas
- HTML
- CSS
- JavaScript


## Cómo ejecutar
1. Descargar o clonar el repositorio
2. Abrir el archivo `index.html` en el navegador
